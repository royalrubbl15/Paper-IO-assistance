# Paper-IO-assistance
Zoom hack, game speed change hack, instant movement hack, and pause/resume functionality. Works for Teams Mode, Classic Mode, Battle Royale Mode, and World Map Mode.
Use Space to freeze the game, W(up), A(left), S(down), D(right) to teleport in that direction, and use the Up Arrow and Down Arrow keys to speed/slow down time. PLEASE DO NOT USE THIS IN A PUBLIC GAME. PLEASE USE THIS IN A PRIVATE GAME, OR AS A FUN SCRIPT TO TEST ON FRIENDS.
